# Polaris 视频工坊 · 可视化链路

```
┌─────────────┐
│   用户文案   │  ← 粘贴/上传课件、文章、口播稿
│ (课件/文章) │
└──────┬──────┘
       │ Phase 1: AI 执行
       ▼
┌─────────────┐
│  口播稿 +   │  ← 口语化改写、加停顿标记
│   outline   │  ← 章节切分、步数、信息池
└──────┬──────┘
       │ Phase 2: AI 执行
       ▼
┌─────────────┐
│  Vite+React │  ← 脚手架 + 逐章开发
│  网页演示   │  ← .tsx + .css + narrations.ts
└──────┬──────┘
       │ Phase 3: 脚本自动 (~2min)
       ▼
┌─────────────┐
│  MiniMax    │  ← T2A 逐段合成
│   T2A 配音  │  ← 13 段 mp3
└──────┬──────┘
       │ Phase 4: 脚本自动 (~3min)
       ▼
┌─────────────┐
│  Playwright │  ← 无头截图 1920×1080
│  逐帧对齐   │  ← 每步精确时长 = 音频时长
└──────┬──────┘
       │
       ▼
┌─────────────┐
│   ffmpeg    │  ← 拼接合成
│   MP4 输出  │  ← H.264 + AAC
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  output.mp4 │  ← 1920×1080 · 音画同步
└─────────────┘
```

## 链路特点

| 特性 | 说明 |
|---|---|
| **4 Phase** | Phase 1-2 由 AI 执行（创意），Phase 3-4 由脚本执行（确定性） |
| **零手动录屏** | Phase 4 用 Playwright 无头截图 + ffmpeg 拼接，无需用户手动操作 |
| **音画同步** | 每步画面冻结时长 = 对应音频精确时长（ffprobe 读取），天然对齐 |
| **一次出片** | 用户只需提供文案，其余全自动化 |

## 各 Phase 耗时参考

| Phase | 内容 | 耗时 | 执行者 |
|---|---|---|---|
| 1 | 文案 → 口播稿 + outline | 1-2 min | AI |
| 2 | 脚手架 + 逐章开发 | 5-10 min | AI |
| 3 | MiniMax T2A 配音 | 1-2 min | 脚本 |
| 4 | 截图 + ffmpeg 合成 MP4 | 2-3 min | 脚本 |
| **总计** | | **~15 min** | |

## 关键文件

- `scripts/install-deps.mjs` — 依赖检查与安装指引
- `scripts/run.mjs` — 一键执行入口
- `scripts/pipeline/03-record.mjs` — Phase 4 录屏合成
