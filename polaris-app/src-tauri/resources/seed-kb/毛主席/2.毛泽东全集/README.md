# 2.毛泽东全集

> 收录《毛泽东全集》1-52 卷(参 [Prolet 仓库 A4 目录](https://github.com/ProletRevDicta/Prolet))。

## 范围

毛泽东全集是迄今为止收录最全的毛泽东著作集,包括:
- 已收入选集的著作
- 各时期的演讲、报告、谈话
- 书信、批示、题词
- 文章、电报、命令

## 建议组织方式

按卷次建子目录:
```
2.毛泽东全集/
├── 第01卷/  ← 1898-1923
├── 第02卷/  ← 1923-1925
├── ...
└── 第52卷/  ← 1976
```

或按年代分:
```
2.毛泽东全集/
├── 1893-1920_早年/
├── 1921-1927_建党与早期革命/
├── 1927-1937_土地革命/
├── 1937-1945_抗日战争/
├── 1945-1949_解放战争/
├── 1949-1956_建国与改造/
├── 1956-1966_探索时期/
└── 1966-1976_文革时期/
```

实际导入时根据来源决定。

## 与选集的关系

- 选集是经过精选和修订的版本(经过编辑加工)
- 全集是按时间顺序的完整收录(尽量保留原貌)
- 同一篇文章在两套版本中可能有细微差异——以全集为原貌底本,以选集为通行本
- 摄入时标注 `version: 全集` 或 `version: 选集` 区分
