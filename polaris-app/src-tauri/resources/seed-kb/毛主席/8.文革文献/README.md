# 8.文革文献

> 收录 1966-1976 文化大革命时期的文献、最高指示。

## 范围与立场声明

本目录收录历史文献的目的是**研究历史**,所有文献应忠实保留原貌,不作删改。
对文献的解读、评价请放入 `wiki/outputs/`,不与原文混存。

## 建议子目录

```
8.文革文献/
├── 1.中央文件/          ← 五·一六通知、十六条等
├── 2.最高指示/          ← 毛主席语录、关于文革的指示
├── 3.重要讲话/          ← 接见红卫兵讲话、九大、十大政治报告(若由毛主持)
├── 4.接见外宾/          ← 文革期间的外交谈话
├── 5.批林批孔与反击右倾翻案风/ ← 相关讲话和指示
└── 6.晚年诗作与谈话/     ← 1970 后的诗词和讲话
```
