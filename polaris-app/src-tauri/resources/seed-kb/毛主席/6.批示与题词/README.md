# 6.批示与题词

> 收录毛泽东的批示、题词、读书笔记。

## 建议子目录

```
6.批示与题词/
├── 1.文件批示/           ← 对中央、地方文件的批示
├── 2.报告批示/           ← 对调查报告、汇报的批示
├── 3.题词/               ← 为机构、报刊、个人的题词
├── 4.读书笔记/           ← 读古书、马列著作的笔记和批注
└── 5.讲话整理稿批示/     ← 对讲话整理稿的修改批示
```

## 重要批示(参考)

- 关于人民内部矛盾的批示
- 关于「七二六」指示
- 对各地报告的批示(《中国农村的社会主义高潮》大量批语)
- 关于「鞍钢宪法」的批示

## 重要题词(参考)

- 为人民服务(1944,在张思德追悼会)
- 自己动手,丰衣足食(1939)
- 团结、紧张、严肃、活泼(抗大校训)
- 向雷锋同志学习(1963-03-05)
- 提高警惕,保卫祖国(1953)
- 备战、备荒、为人民(1965)

## 重要读书笔记(参考)

- 读《政治经济学(教科书)》笔记
- 读《资治通鉴》《二十四史》笔记
- 读列宁《国家与革命》笔记
- 读《共产党宣言》笔记
