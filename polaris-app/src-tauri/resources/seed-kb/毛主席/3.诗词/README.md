# 3.诗词

> 收录毛泽东各时期诗词原文。

## 建议子目录

按创作时期分:
```
3.诗词/
├── 1.早期(1900-1925)/         ← 五古·挽易昌陶、贺新郎·别友 等
├── 2.大革命与井冈山(1925-1929)/ ← 沁园春·长沙、西江月·井冈山 等
├── 3.长征时期(1934-1936)/      ← 七律·长征、清平乐·六盘山、念奴娇·昆仑 等
├── 4.延安抗战(1937-1945)/      ← 沁园春·雪、五律·挽戴安澜 等
├── 5.解放战争(1945-1949)/      ← 七律·人民解放军占领南京 等
├── 6.建国初期(1949-1957)/      ← 浣溪沙·和柳亚子先生、水调歌头·游泳 等
├── 7.大跃进与调整(1957-1965)/  ← 蝶恋花·答李淑一、七律·到韶山 等
└── 8.晚年(1966-1976)/         ← 七律·有所思、读《封建论》呈郭老 等
```

```

示例:
沁园春_雪_1936-02.md
七律_长征_1935-10.md

```

每首诗词建议在文末附:
- 创作背景
- 注释(地名、典故)
- 与同期文章/事件的关联
