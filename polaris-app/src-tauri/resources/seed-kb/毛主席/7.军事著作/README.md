# 7.军事著作

> 收录毛泽东的军事著作、电报、作战命令。

## 范围

虽然部分军事著作(如《中国革命战争的战略问题》《论持久战》)已收入选集,但毛泽东的军事文献量极大,且不少电报、命令未收入选集——单设此目录便于军事思想的整体研究。

## 建议子目录

```
7.军事著作/
├── 1.战略论著/           ← 中国革命战争的战略问题、论持久战、战争和战略问题等
├── 2.战术论著/           ← 十大军事原则、关于辽沈战役的指挥等
├── 3.作战指挥电报/       ← 红军时期、抗日时期、解放战争时期的电报
├── 4.作战命令/           ← 重大战役的命令文件
└── 5.军队建设/           ← 古田会议决议、政治工作、军队整顿
```

## 与选集的去重处理

如选集和军事著作目录都有同一篇,选择:
- 在 `1.毛泽东选集/` 放选集版本
- 在 `7.军事著作/` 用 [指针文件] 链接到选集版本
- 或反之,以最先入库为正本

详见根目录 schema.md 的指针文件规范。
