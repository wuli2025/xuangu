# 4.演讲与谈话

> 收录重要讲话、报告、与外宾谈话、内部讲话记录。

## 建议子目录

按时期分:
```
4.演讲与谈话/
├── 1.建党与早期(1921-1927)/
├── 2.土地革命(1927-1937)/
├── 3.延安时期(1937-1945)/    ← 整风讲话、文艺座谈会等
├── 4.解放战争(1945-1949)/
├── 5.建国初期(1949-1956)/    ← 七届二中全会、中共八大开幕词等
├── 6.探索时期(1956-1966)/    ← 论十大关系、人民内部矛盾等
└── 7.文革时期(1966-1976)/    ← 接见外宾谈话、内部讲话
```

或按类型分:
```
4.演讲与谈话/
├── 重大会议讲话/  ← 七大、七届二中全会、八大、九大、十大等
├── 文艺与教育/    ← 延安文艺座谈会、文教界谈话
├── 接见外宾/      ← 与各国领导人、知识分子的谈话
└── 内部讲话/      ← 中央会议、调查谈话
```
