## 文件命名规范

```
<著作名/类别>_<版本或时间>.md

示例:
矛盾论_1937.md
沁园春_雪_1936-02.md
致毛岸英_1947-10-08.md
```

详见根目录 `schema.md`。

## 摄入流程

1. 将文件放入对应分类子目录(必要时新建子目录如「第一卷」「延安时期」)
2. 通知 LLM 执行模式 4(Ingest 工作流)
3. LLM 先做分析报告,等用户确认后再编译写入 wiki/

**严禁**:
- 直接修改 raw/ 中文件内容(发现 OCR 错误等,在 wiki/ 中标注)
- 在 raw/ 中放分析、解读、摘要等加工后内容(那些放 wiki/)
- 在 raw/ 中放二手转述、后人作品
