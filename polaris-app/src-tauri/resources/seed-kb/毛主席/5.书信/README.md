# 5.书信

> 收录毛泽东往来书信。

## 建议子目录

```
5.书信/
├── 1.家书(致亲属)/       ← 致毛岸英、毛岸青、杨开慧等
├── 2.致同志/             ← 致彭德怀、林彪、周恩来、刘少奇等
├── 3.致党外人士/         ← 致柳亚子、章士钊、傅作义等
├── 4.致友人/             ← 致老师、同学、故旧
└── 5.复信(对群众)/      ← 复工人、农民、青年学生的信
```

## 重要书信(参考)

- 致毛岸英(关心儿子学习与劳动)
- 致彭德怀(战争年代军事指挥)
- 致黄炎培(关于「窑洞对」延伸讨论)
- 致柳亚子(诗词唱和与时局)
- 致周世钊(老同学、故旧之情)
- 致李达(早期理论交流)
- 致陈毅(谈诗与谈思想)

## 文件命名

```
致XXX_主题_YYYY-MM-DD.md

示例:
致毛岸英_关于劳动锻炼_1947-10-08.md
致柳亚子_答和先生韵_1949-04-29.md
致彭德怀_关于平江起义_1959-XX.md
```

如无具体日期,可用 `YYYY-XX.md` 或 `YYYY-MM.md`。
